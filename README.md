# Plan2Do

Plan2Do is a  simple notepad app, built with Kotlin using Room DB,Hilt,Fragments
LiveData, and following Model-View-ViewModel (MVVM).
## Features

- Light/dark mode toggle
- SET PRIORITIES
- Update Note
